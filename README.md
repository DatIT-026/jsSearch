# TRƯỜNG SĨ QUAN TĂNG THIẾT GIÁP

## Introduction
A website designed to manage **student/cadet records for the Trường Sĩ Quan Tăng Thiết Giáp** effectively. The system provides quick access and management capabilities for student personnel data.

---

## Features
- **Efficient management** and resolution of all matters related to **student/cadet personnel records**, including processing, updating, and maintaining accurate data.
- **Quick Search Functionality** for student information using a formal **Electronic Information Portal** interface.


You can view online demo [here](https://datit-026.github.io/jsSearch/).
